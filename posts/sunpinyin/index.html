修改了Sunpinyin的全角快捷键

Sunpinyin的全角快捷键默认为opt+space
我经常会误按这个组合，导致输入变为全角，感觉很不方便
因此下了源码自己修改了一下快捷键，同时修改新图标为以前的图标


首先是参考的教程，如何在maces下build SunPinyin 2.0

http://code.google.com/p/sunpinyin/wiki/BuildSunpinyin2onMac


下面是我操作的步骤

1. 安装scons
下载http://prdownloads.sourceforge.net/scons/scons-1.3.1.tar.gz，并将其解压，
$ cd scons-1.3.1
$ sudo python setup.py install

2. 从github上clone/pull最新的sunpinyin源代码
$ git clone git://github.com/sunpinyin/sunpinyin.git

3. 修改图标, 原有图标需要自行下载
$ cp old-sunpinyin/SunPinyin.icns sunpinyin/wrapper/macos/

4. 修改快捷键
使用Xcode打开 sunpinyin/wrapper/macos/SunPinyin.xcodeproj
打开 工程中的 Resources/MainMeneu.xib
在编辑页面左侧选择 Menu, 然后在出现的Menu中,双击全角切换右侧的快捷键
此时就可以自行设置快捷键, 我的选择是直接按CMD取消了此处的快捷键
不要忘了保存

5. 使用scons来构建sunpinyin的工具及数据文件
$ cd sunpinyin
$ scons

这里在编译时, 会下载几个资源文件。网络不稳定容易下载失败, 此时可以手动将链接复制到浏览器,进行下载。
然后将下载后的文件放在 sunpinyin/build目录下
重新运行scons命令就可以了

6. 最后就是build基于IMKit的输入法前端了，
$ cd wrapper/macos
$ make clean; make
$ sudo make install

现在，sunpinyin-2.0应该已经安装到系统中了，需要re-login才可以看到这个输入法。
